The Scan WP Theme and Plugin Detector is a free detector you can use on your own site in the admin area. Detect other sites' themes and plugins.

== Description ==

This plugin is the Wordpress plugin version of the known [Scan WP](http://scanwp.net "Scan WP") Theme and plugin detector.
This plugin is for your use only since it will be used in the admin section of your site.
If you, like many people have a hard time choosing a theme for your site or knowing what plugins to use out of the bunch,
all you need to do is find a site that you like, add the URL in your very own theme detector in the admin area and hit "search".

The results will contain the theme name, price, version, download link, plugin names, plugin price and download links.

Now all you have to do is decide what you want to install and you are good to go!

= Scan WP Theme and Plugin Detector - Admin Version support =
Any questions regarding the plugin will be answered and handeled here: <a href="http://scanwp.net/our-plugins">Scan WP Plugins</a>

== Installation ==

The best practice to install the plugin is:

1. Download the plugin
2. In your admin panel, go to Plugins -> Add New
3. Click "Upload Plugin" and choose the Zip file you downloaded
4. After the plugin is uploaded click "Activate plugin"

If you rather do things manually:

1. Upload 'scan-wp-theme-and-plugin-detector-admin-version' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
